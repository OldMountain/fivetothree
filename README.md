## 前端

+ 自定义多选框/单选框
```
<div class="ftt-checkbox">
    <label for="$id$">
        <input type="checkbox" name="" id="$id$">
        <i></i>
            选项1
    </label>
</div>
```
+ 自定义开关按钮
```
<div class="ftt-switch">
    <input type="checkbox" name="$id$"
           id="$id$">
    <label for="$id$"></label>
    <i><span>on</span><span>off</span></i>
</div>
```
